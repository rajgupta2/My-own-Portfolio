## 📌 About

This repository powers my personal portfolio site — a single place to explore who I am as a developer, what I've built, and how to get in touch.

I'm a **Gold Medalist** B.Tech (Information Technology) graduate, with hands-on internship experience in both full stack development and cloud infrastructure automation.

## ✨ Features

- 🎬 **Animated Hero section** with a dynamic typing effect introducing my roles
- 📖 **About section** highlighting my background, certifications, and current focus
- 🛠️ **Skills grid** with hover animations, covering my full MERN + AWS/DevOps stack
- 💼 **Projects showcase** with live preview toggles, GitHub links, and tech stack tags
- 📬 **Contact section** with social links and embedded certification badges
- 📱 **Fully responsive** — optimized layouts from mobile to desktop
- 🌗 **Consistent theming** across every section with smooth scroll-reveal animations

## 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/rajgupta2/My-own-Portfolio.git
```

Move into the project directory

```bash
cd My-own-Portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser and visit

```
http://localhost:3000
```
